About DeepMedic.

DeepMedic is BioMedIA's (Biomedical Image Analysis Group, Department of Computing, Imperial College London) software for 3D image segmention, based on a multi-scale 3D Deep Convolutional Neural Network coupled with a 3D fully connected Conditional Random Field. The system has been shown to yield excellent performance on brain lesion segmentation (winner of the ISLES 2015 competition) on challenging lesion segmentation tasks, including traumatic brain injuries, brain tumors, and ischemic stroke lesions.

3D Deep Learning Suite

A complete set of versatile and easy to use tools to train, evaluate, and make use of 3D Image Segmentation Models in a simple and intuitive manner for non-technical users.

Graphical Interface

DeepMedic's Graphical User Interfaces enable an easy use of its powerful Deep Learning framework to easily train accurate 3D segmentation models.

Integrated Image Pre-Processor

Automated data harmonisation, and informative data scans, used via a Graphical User Interface.

Tried and Tested Performance

DeepMedic's the winning algorithm for the ISLES 2015 competition, and part of the winning framework for BraTS 2017. It is also widely used in the medical field to deliver state-of-the-art performance.

Easy To Configure And Customize

DeepMedic's informative graphical interfaces allow for an easy tuning of Deep Learning model parameters to achieve higher performance levels.

Pre-Trained Model Zoo

DeepMedic distributes pre-trained 3D Image Segmentation models as well fine-tuned pre-processing configurations, for common medical tasks.
